## Food Delivery System - Microservices Architecture

This repository contains a production-ready microservices-based **Food Delivery System**.

- **Backend**: Java 17, Spring Boot 3.x, Spring Cloud
- **Services**: Eureka Server, API Gateway, User, Restaurant, Order, Payment, Delivery, Notification
- **Database**: MySQL (separate schema per service)
- **Messaging**: Apache Kafka
- **Auth**: JWT-based authentication with role-based authorization
- **Frontend**: React (Vite)
- **DevOps**: Docker, Docker Compose

See `docs/RUNNING.md` for detailed setup and run instructions.
